# 2. Setup and Starting State

Finish setup before any round begins.

## 2.1 Required Components

Prepare these components before you start:
- 2 players
- 1 skill deck with 40 cards for each player
- 3 hero cards for each player
- 4 arcane items
- 1 arcane field
- 1 5x5 arena

## 2.2 Build the Board

Set up the board in this order:
1. Create the 5x5 arena.
2. Place 1 random arcane field face down on the center tile.
3. Place 4 random arcane items face down on the 4 corner tiles.

> **What This Means:** The center is always the final objective space, and the corners are always the starting locations of the collectible objective pieces.

## 2.3 Place Heroes

Place each player's 3 heroes in that player's hero pool zone.
The hero pool zone is outside the arena.
A hero in the hero pool zone does not occupy a tile.

## 2.4 Set Starting States

At the start of the game, each hero is in its hero pool zone, not deployed, ready, not dead, and has 0 taken damage and a death timer of 0.
Each player's wisdom (max hand size) is 3 unless another rule says otherwise.
